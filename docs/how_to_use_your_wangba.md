# 上传

在克隆下来的根目录下右键，找到乌龟，先`commit`。这里你`commit`到哪个分支都行，因为都是你电脑上的分支。

然后`commit`完王八会让你`push`，注意这时候要`push`到一个新的分支，因为这里已经是远程分支了，看图。
